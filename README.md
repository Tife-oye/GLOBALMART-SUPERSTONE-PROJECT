### Project Overview
This project analyzes the Global Superstore dataset using Power BI to uncover valuable insights into business performance, 
profitability, sales trends, and supply chain operations. 
The goal is to transform raw business data into actionable insights that support strategic decision-making.

# Key Metrics
•	Total Sales

•	Total Revenue

•	Total Profit

•	Profit Margin

•	Number of Orders

•	Delivery Performance

•	Sales Growth

•	Revenue Trend

•	Regional Performance

•	Supplier Performance

# Tools Used
•	Microsoft Power BI

•	Power Query

•	DAX (Data Analysis Expressions)

•	Data Modeling

•	Data Cleaning & Transformation


# Business Recommendations
•	They should invest more in high-profit products and categories.

•	They should strengthen partnerships with reliable suppliers.

•	They should improve logistics to reduce delivery delays.

•	They should expand operations in high-performing regions.

•	They should optimize inventory based on customer demand.
